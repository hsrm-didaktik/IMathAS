Macro Library chatter

A macro library providing tools for communicating with chatbots

June 2024
  • asciimath2tex
  • copyButton

Background

Chatbots work best with text prompts in Markdown format with formulas in LaTeX. The tools in this library simplify the process of encoding information from randomized question in an appropriate format and transferring it to a chatbot.

asciimath2tex($arg)

This function converts a formula in ASCIIMath to LaTeX.

copyButton($arg [, $label, $successMessage, $failMessage, $formatPrompt])

This function returns a button with label that copies arg to the clipboard and reports in an alert successMessage respectively failMessage in case of failure. formatPrompt is prepended to the string to be copied. By default it contains information about ASCIIMath in German.

Notes:

Strings copied from answerbox fields - except for essay fields - should be enclosed in backticks `...`.

Special characters outside of formulas may be replaced by the respective HTML entities when copied from an essay field, For example < will be replaced by &lt;.
