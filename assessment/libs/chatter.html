A macro library providing tools for communicating with chatbots

Background

Chatbots work best with text prompts in Markdown format with formulas in LaTeX. The tools in this library simplify the process of encoding information from randomized question in an appropriate format and transferring it to a chatbot.

For transmission as URL parameters, strings should be URL-encoded. To URL-encode a string use the function rawurlencode.

asciimath2tex(arg)

This function converts a formula in ASCIIMath to LaTeX. arg should be an ASCIIMath formula without enclosing backticks. The result will be enclosed between $-signs.

essayText(arg)

The function takes an HTML string arg as argument as it is returned as a student answer from an essay question. It removes all HTML tags and replaces ASCIIMath formulas with LaTeX formulas.

chaturl(model, prompt [, openwebuiBaseUrl])

This function builds a URL for a chat on Open WebUI. It takes the following parameters:

  • model: the model to use for the chat
  • prompt: the prompt to use for the chat. The prompt must be formatted in markdown with formulas in LaTeX. It must not be URL-encoded, as it will be encoded by the function.
  • openwebuiBaseUrl: the base URL of the OpenWebUI instance to use. Defaults to https://chatter.dahn-research.de.

Use as $url=chaturl($model, $prompt).

url2openwebui(arg [, openwebuiBaseUrl])

This function takes a URL arg for a chat in https://chatsite.dahn-research.de with urlencoded parameters as argument and returns a URL that opens the chat in the Open WebUI instance referenced in openwebuiBaseUrl which defaults to https://chatter.dahn-research.de. If necessary, the function extracts parameters from the URL and constructs a new prompt that uses these parameters.

Use as $url=url2openwebui($url).

copyButton(arg [, label, successMessage, failMessage, formatPrompt])

This function returns a button with label that copies arg to the clipboard and reports in an alert successMessage respectively failMessage in case of failure. formatPrompt is prepended to the string to be copied.

Notes:

Strings copied from answerbox fields - except for essay fields - should be enclosed in backticks `...`.

Special characters outside of formulas may be replaced by the respective HTML entities when copied from an essay field, For example < will be replaced by &lt;.
